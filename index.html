Project Overview

WorldVLN is presented here as an anonymous research project page for blind review. The goal of this website is to provide a clean, reviewer-friendly summary of the project and a centralized location for supplementary qualitative videos.

Following the structure of a typical academic project website, the page focuses on the most useful review-time elements: a concise overview, key characteristics, direct access to demo videos, and anonymized repository information. Identity-bearing content is intentionally excluded.

Anonymous review notice: this page keeps only project-essential information online. Author details, institutional logos, personal profiles, and post-acceptance materials can be added later.
